Implementation:

1. Local state: const [value, setValue] = useState('')
2. @claude detection: const isClaudeTagged = value.includes('@claude')
3. Submit handler:
   ```typescript
   const handleSubmit = () => {
     if (!value.trim() || !prKey) return;
     postUserRequest(prKey, {
       type: 'inline_comment',
       payload: {
         lineId,
         message: value.trim(),
         isClaudeTagged,
       },
     }).catch(() => {/* error handled by toast in App.tsx */});
     onClose();
   };
   ```

   Both @claude-tagged and non-tagged comments go through /api/user-request with type 'inline_comment'. The server (Plan 02) handles the branching: it always fires thread.userStarted event, but only enqueues to the LLM queue if isClaudeTagged === true.

4. Keyboard: Enter submits (unless Shift held). Escape closes.
   ```typescript
   const handleKeyDown = (e: React.KeyboardEvent) => {
     if (e.key === 'Enter' && !e.shiftKey) {
       e.preventDefault();
       handleSubmit();
     }
     if (e.key === 'Escape') {
       onClose();
     }
   };
   ```
