| ASVS Category | Applies | Standard Control |

| V2 Authentication | no | ‚Äî |

### Known Threat Patterns

| Pattern | STRIDE | Standard Mitigation |
|---------|--------|---------------------|
| Prompt injection via chat message | Tampering | Chat `message` rendered as React text node, never innerHTML |
| XSS via LLM-authored chat responses | Tampering | `chat.llmMessage.message` rendered as React text node ‚Äî NEVER `dangerouslySetInnerHTML` |
| CSRF on `/api/user-request` | Tampering | Double-submit token inherited from Phase 1 |
| Queue flooding | Denial of Service | Max queue depth (10 items) in `SessionRequestQueue` |
| lineId forgery in inline comment | Tampering | Server resolves via `resolveLineIdExtended`; invalid returns 400 |
