<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Server ShikiToken → DiffViewer innerHTML | Color hex + content string flow through inline style + text into `dangerouslySetInnerHTML`. Content must be HTML-escaped to defend against a misbehaving Shiki (hypothetical); color value is whitelisted to hex-only via a runtime regex check |
| GitHub comment body (ReadOnlyComment.body) → DiffViewer render | Arbitrary user-authored text; MUST render as React text nodes, never innerHTML |
| onExpandGenerated / onMarkReviewed callbacks | Invoked with trusted fileId from local state; no user input |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-3-01 | Tampering | `dangerouslySetInnerHTML` rendering Shiki tokens | mitigate | `tokenToHtml` uses `escapeHtml` on the token content BEFORE wrapping in `<span style="color:{color}">`. Color is validated to match `/^#[0-9a-f]+$/i` OR omitted entirely (defense in depth; Shiki never produces non-hex colors, but the check protects against future regressions). Test asserts that a token with `content: '<script>alert(1)</script>'` renders the literal text, not an executable script. |
| T-3-03 | Tampering | `ReadOnlyComment.body` rendering | mitigate | Comment body renders exclusively through `{comment.body}` inside JSX (React text node). NEVER passed to `dangerouslySetInnerHTML`. Test asserts that a comment with `body: '<img src=x onerror=alert(1)>'` renders as literal text in the popover; no alert fires. |
| T-3-01a | Tampering | Color style attribute via Shiki token | mitigate | Color value validated to hex-only pattern before interpolation; on failure, omit the style attribute. Prevents `color: red; background: url(javascript:alert(1))` type injection. |
</threat_model>

    ```typescript
    function escapeHtml(s: string): string {
      return s
        .replace(/&/g, '&amp;')
        .replace(/</g, '&lt;')
        .replace(/>/g, '&gt;')
        .replace(/"/g, '&quot;')
        .replace(/'/g, '&#39;');
    }

    // Validate hex color before interpolation (T-3-01a defense in depth).
    const HEX_COLOR = /^#[0-9a-fA-F]{3,8}$/;

    function tokenToHtml(tokens: ShikiToken[]): string {
      return tokens.map(tok => {
        const styles: string[] = [];
        if (tok.color && HEX_COLOR.test(tok.color)) {
          styles.push(`color:${tok.color}`);
        }
        if (tok.fontStyle) {
          if (tok.fontStyle & 1) styles.push('font-style:italic');
          if (tok.fontStyle & 2) styles.push('font-weight:bold');
        }
        const safe = escapeHtml(tok.content);
        if (!styles.length) return safe;
        return `<span style="${styles.join(';')}">${safe}</span>`;
      }).join('');
    }
    ```
