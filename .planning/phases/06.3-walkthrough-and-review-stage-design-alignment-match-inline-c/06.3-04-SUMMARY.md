## Decisions Made

- Wrapped the diff column in a flex-column `<div>` in App.tsx to slot WalkthroughStepBanner above DiffViewer without requiring CSS grid changes. The outer `main--diff` grid layout (240px 1fr 360px) still controls column widths; the wrapper just enables vertical stacking within the 1fr column.
- `allFindings` prop added to thread HunkProps to allow gutter markers to display globally-unique 1-based indices (not per-file indices).
- `!important` used on `diff-line--finding-*` background classes to ensure they appear even when the row already has an `add`/`rem` background, keeping the tint subtle at opacity 0.06.
- Invalid finding collapse is a structural DOM change (flex row with DISMISSED label) not just opacity — this matches the design spec D-14 requirement that "the card collapses to a single dismissed row."
