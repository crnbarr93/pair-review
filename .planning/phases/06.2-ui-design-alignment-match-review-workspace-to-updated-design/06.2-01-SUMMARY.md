# Phase 06.2 Plan 01: Store Extensions + CSS Foundation + Container Components Summary

Foundation layer complete: `activeStep` store state with step-based CSS grid and three new container components (RightPanel, StepFooter, SummaryStep) that Plans 02-04 wire into the App shell.

## What Was Built

### Store Changes (web/src/store.ts)
- Added `activeStep: 'summary' | 'walkthrough' | 'review' | 'submission'` to `AppState` interface
- Added `activeStep: 'summary'` to `INITIAL` (default to summary step)
- Added `setActiveStep(step)` action following the existing `setChatPanelOpen` spread+emit pattern
- No existing actions, `onSnapshot`, `onUpdate`, or subscription patterns modified

### RightPanel.tsx (new)
- Container with two slots: `children` (step-specific content, top) and `chatSlot` (ChatPanel, bottom)
- CSS classes: `.right-panel` → `.right-panel-content` (flex 1) + `.right-panel-chat` (max-height 50%, border-top)
- Slot pattern avoids circular imports — App.tsx passes the ChatPanel instance as `chatSlot`

### StepFooter.tsx (new)
- Contextual footer bar rendering per-step actions
- Summary: Regenerate summary button + last-generated age + Continue to walkthrough CTA
- Walkthrough: Mark all reviewed + Continue to review CTA
- Review: Export as comments + Continue to submission CTA
- Submission: returns `null` (CTA lives in right panel per D-10)
- Exports `ReviewStep` type for consumer use

### SummaryStep.tsx (new)
- Full-page summary view promoted from SummaryDrawer drawer pattern
- Stats grid (4 cards): files changed, lines added, lines removed, commits (PR placeholder)
- Tab bar: "Claude's summary AUTO" | "Author's description ORIGINAL" — local useState
- Claude tab: intent chip, confidence %, paraphrase, key changes, risk areas — each section has an "Ask" button
- Ask buttons call `postUserRequest(prKey, { type: 'chat', payload: { message } })` to open chat
- Author tab: renders `authorDescription` in `.summary-author-block` or "No description provided"

### CSS Changes (web/src/index.css)
- App grid: `grid-template-rows: auto 1fr 44px` (was `44px 52px 1fr`)
- New `.main--summary` class: `grid-template-columns: 1fr 360px`
- New `.main--diff` class: `grid-template-columns: 240px 1fr 360px`
- Removed `.main--panel-open` and `.main--findings-open` (superseded)
- Added `.topbar-shell` for two-row TopBar wrapper
- Added `.right-panel`, `.right-panel-content`, `.right-panel-chat` sections
- Added `.step-footer`, `.footer-secondary`, `.footer-spacer`, `.footer-cta` classes
- Added full `.summary-step`, `.summary-stats-grid`, `.summary-stat-card`, `.summary-tabs`, `.summary-tab`, `.summary-section`, `.summary-section-header`, `.summary-ask-btn`, `.summary-author-block` suite
- Added `.chat-context-badge` for per-step context display
- Removed `.footer-hint` rule per D-11 (keyboard hint bar dropped entirely)
- Updated `.findings-sidebar` from `height: 100%` to `flex: 1; overflow-y: auto` for inline flex use
- Removed fixed-position media query for `.findings-sidebar` (now inline in right panel)
