**StepFooter.tsx (new file):**

Create a contextual footer bar. Receives the active step and step-specific callbacks.

```typescript
export type ReviewStep = 'summary' | 'walkthrough' | 'review' | 'submission';

interface StepFooterProps {
  step: ReviewStep;
  onRegenerateSummary?: () => void;
  onMarkAllReviewed?: () => void;
  onExportComments?: () => void;
  onContinue?: () => void;
  summaryAge?: string;          // e.g. "2m ago"
  walkthroughStepsVisited?: number;
  walkthroughStepsTotal?: number;
}
```

Render per step:
- summary: left = "Regenerate summary" button + "Last generated {summaryAge}" text, right = "Continue to walkthrough" CTA button with arrow
- walkthrough: left = "Mark all reviewed" button, right = "Continue to review" CTA button with arrow
- review: left = "Export as comments" button, right = "Continue to submission" CTA button with arrow
- submission: return null (no footer — CTA is in right panel per D-10)

Use CSS class `step-footer` for the bar, `footer-secondary` for left-side actions, `footer-cta` for right-side primary CTA. The CTA button uses the same `.primary` class as the TopBar's submit button.
