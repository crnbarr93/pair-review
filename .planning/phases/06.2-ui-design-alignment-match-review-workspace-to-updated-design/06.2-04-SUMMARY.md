# Phase 06.2 Plan 04: App.tsx Integration — Step-Routed App Shell Summary

App.tsx rewritten as a 4-step routed shell using all components from Plans 01-03. WalkthroughBanner inline render removed from DiffViewer. Human pixel-match verification pending.

## What Was Built

### Task 1: App.tsx step-routing rewrite (commit `0b5bf49`)

Complete rewrite of App.tsx to wire together all Phase 06.2 foundation components:

- **Step routing** via `state.activeStep` from store: `summary | walkthrough | review | submission`
- **Summary step** (`main--summary` grid): renders `SummaryStep` + `RightPanel` with chat only (no diff, no file explorer)
- **Diff steps** (`main--diff` grid): renders `FileExplorer` + `DiffViewer` + `RightPanel` with step-specific content above chat
  - Walkthrough: `WalkthroughStepList` in RightPanel children
  - Review: `FindingsSidebar` in RightPanel children
  - Submission: `SubmissionPanel` in RightPanel children
- **StepFooter** rendered below main for all steps except submission
- **ChatPanel** extracted to a shared `chatPanelSlot` variable — passed as `chatSlot` prop to `RightPanel` in all branches
- **chatContextBadge** computed via `useMemo` per step + focused file/hunk (D-04)
- **chatSuggestionChips** computed via `useMemo` per step with contextual suggestions (D-04)
- **handleStepClick** wired to `TopBar.onStepClick`; **handleContinue** wired to `StepFooter.onContinue`
- **handleRegenerateSummary** calls `postUserRequest(prKey, { type: 'regenerate_summary' })`
- **handleMarkAllReviewed** iterates diff files, posts `file.reviewStatusSet` for non-reviewed files
- **Keyboard shortcuts** `v` and `s` updated to navigate to submission step instead of opening modal

**Removed from old App.tsx:**
- `StageStepper` component render
- `SummaryDrawer` conditional render
- `summaryDrawerOpen` local state
- `FindingsSidebar` and `ChatPanel` rendered inside `<main>` (moved to RightPanel)
- `.footer-hint` div (D-11 keyboard hint bar removed)
- `app--findings-open` class on root div
- `main--panel-open` and `main--findings-open` classes on `<main>`

All existing event handlers preserved unchanged (advanceHunk, markCurrentFileReviewed, handleWalkthroughStepClick, handleWalkthroughStepComplete, handleWalkthroughStepToggle, handleShowAllToggle, handleSkipStep, handleNextStep, handleDraftChange, handlePickFile, handleExpandGenerated, handleMarkReviewed).

### Task 2: WalkthroughBanner removal (commit `7289625`)

- Removed `import { WalkthroughBanner }` from `DiffViewer.tsx`
- Removed `<WalkthroughBanner>` JSX block from the hunk render loop
- Kept `walkthroughStepForHunk` and `stepIndex` variables — still needed for `hunk--curated` CSS class
- Added DEPRECATED comment to `WalkthroughBanner.tsx`; file retained for reference
