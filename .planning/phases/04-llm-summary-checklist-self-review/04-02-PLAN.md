| Boundary | Description |

| TS const → tool description (Plan 04-06) | CHECKLIST text lands in run_self_review's `description` string at server-start time. The Claude Code session reads it via tools/list. Supply-chain angle: this const ships in the bundled server code; tampering requires code edit, not runtime injection. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
