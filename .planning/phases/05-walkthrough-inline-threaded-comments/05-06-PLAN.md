**In `web/src/App.tsx`:**

Add a `filteredDiff` useMemo between the existing `virtualList` useMemo (line ~49) and the `showToast` callback (line ~67). This memo computes the DiffModel that DiffViewer should render:

```typescript
const filteredDiff = useMemo(() => {
  if (!diff) return undefined;
  const wt = state.walkthrough;
  // No walkthrough active or show-all mode: render everything
  if (!wt || wt.showAll) return diff;
  // Curated mode: only hunks in walkthrough.steps
  const curatedHunkIds = new Set(wt.steps.map(s => s.hunkId));
  const filtered: DiffFile[] = [];
  for (const file of diff.files) {
    // Always skip generated files (they are excluded from walkthrough)
    if (file.generated) continue;
    const kept = file.hunks.filter(h => curatedHunkIds.has(h.id));
    if (kept.length > 0) {
      filtered.push({ ...file, hunks: kept });
    }
  }
  return { files: filtered, totalHunks: filtered.reduce((n, f) => n + f.hunks.length, 0) };
}, [diff, state.walkthrough]);
```

Then change the DiffViewer invocation (currently at line ~413) to pass `filteredDiff` instead of `diff`:

Change:
```tsx
<DiffViewer
  diff={diff}
```

To:
```tsx
<DiffViewer
  diff={filteredDiff ?? diff}
```

Also change the guard condition at line ~402 from `{diff && (` to `{diff && filteredDiff !== undefined && (` -- actually this is simpler: since `filteredDiff` returns `undefined` when `diff` is undefined, keep the outer `{diff &&` guard and just pass `filteredDiff ?? diff` to the DiffViewer prop. The `filteredDiff` is always defined when `diff` is defined (the memo returns `diff` itself in show-all/no-walkthrough cases).

**Important**: Do NOT change the FileExplorer prop. FileExplorer should still receive the full `diff` so the file tree always shows all files regardless of walkthrough filter state. Only DiffViewer gets the filtered view.
